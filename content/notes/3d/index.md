## Steps

1. Open hugo-new and copy contents.
2. Create a new document in [vscode.dev](https://vscode.dev/).
3. Create all the daily draws for the month. match it with the schedule.
4. Add dates with `result-today-august-1` format.
5. Select all and sort/ascending.
6. Duplicate the default arch and rename it to the month.
7. Match game counts.
8. Rename files from vscode.dev.
9. Update x.py.

{{% callout warning %}}
Make sure to update the script directory.
{{% /callout %}}

{{% callout note %}}
Count how many instances so that it matches with the current month.
{{% /callout %}}

{{% callout note %}}
Use the `.md'` as search param. Use alt to unselect/select the top line.
{{% /callout %}}

{{% callout note %}}
Use the batch rename function to maintain alphanumeric order.
{{% /callout %}}

{{% callout note %}}
Use `-(\d)"` regex to find all single-digit dates and replace it with `-0$1"`.
{{% /callout %}}

10. Run x.py `python3 archetypes/month/x.py`.
11. Run `hugo new --kind month archive`.
