## Add title and year established within the main element

```html
<main>
  <h1>CAMPER CAFE</h1>
  <p>Est. 2020</p>
</main>
```

## Add 2 sections (for coffees and desserts) within the main element

```html
<main>
  <h1>CAMPER CAFE</h1>
  <p>Est. 2020</p>
  <section>
  </section>
  <section>
  </section>
</main>
```
