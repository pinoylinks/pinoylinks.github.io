## Steps

1. Change the origin of the object to the bottom by going to edit mode, select bottom face, 3D cursor to selection, origin to 3D cursor.
2. Enable snap mode
3. Open snap mode settings, enable snap individual elements to face project.
4. Enable align rotation to target.
