## Steps

1. Add bunch of spheres in a cloud shape.
2. Select all spheres and join (CTRL + J).
3. Apply scale.
4. Add modifier/generate/remesh/voxel size: .07 m.
5. Add modifier/generate/decimate/ratio: .0125.
6. You can still change the shape of the cloud in edit mode.
7. Apply modifiers.
